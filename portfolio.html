Java
Sistema de Recomendação de Conteúdo
Aplicação desenvolvida em Java que implementa um sistema de recomendação baseado em filtragem colaborativa. O sistema analisa o histórico de interações dos usuários, calcula similaridades entre perfis e sugere conteúdos relevantes com base nas preferências identificadas. Foi meu primeiro projeto completo unindo lógica de algoritmos com programação orientada a objetos.
// trecho do código
// Cálculo de similaridade por cosseno entre usuários public double calcularSimilaridade(Map<String, Integer> perfil1, Map<String, Integer> perfil2) { double dotProduct = 0, norm1 = 0, norm2 = 0; for (String item : perfil1.keySet()) { if (perfil2.containsKey(item)) { dotProduct += perfil1.get(item) * perfil2.get(item); } norm1 += Math.pow(perfil1.get(item), 2); } for (int val : perfil2.values()) norm2 += Math.pow(val, 2); return dotProduct / (Math.sqrt(norm1) * Math.sqrt(norm2)); }
// tecnologias utilizadas
Java 17 OOP Collections Framework Algoritmos de similaridade Maven
// funcionalidades
  • Cadastro e gerenciamento de usuários e conteúdos
  • Algoritmo de filtragem colaborativa (User-Based CF)
  • Cálculo de similaridade por cosseno entre vetores de preferência
  • Geração de lista personalizada de recomendações
  • Exportação dos resultados em formato texto

Automação de Processos Jurídicos
Solução de automação desenvolvida para o escritório de advocacia onde trabalho. O script processa planilhas com dados de processos, organiza documentos em pastas por cliente/processo, gera relatórios de acompanhamento e envia notificações automáticas de prazos por e-mail. Reduziu significativamente o tempo gasto em tarefas administrativas repetitivas.
// trecho do código
import schedule, smtplib from datetime import datetime, timedelta def verificar_prazos(processos, antecedencia_dias=3): alertas = [] hoje = datetime.today() for proc in processos: prazo = datetime.strptime(proc['prazo'], '%d/%m/%Y') if timedelta(0) <= (prazo - hoje) <= timedelta(antecedencia_dias): alertas.append(proc) return alertas schedule.every().day.at("08:00").do(verificar_prazos)
// tecnologias utilizadas
Python openpyxl smtplib os / shutil datetime schedule

from sklearn.feature_extraction.text import TfidfVectorizer from sklearn.metrics.pairwise import cosine_similarity def identificar_intencao(mensagem, intencoes, vectorizer, matriz): vec = vectorizer.transform([mensagem]) scores = cosine_similarity(vec, matriz).flatten() idx = scores.argmax() if scores[idx] > 0.3: return intencoes[idx] return 'desconhecida'
